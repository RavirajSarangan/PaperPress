Add Page Numbers to PDF — Free, No Upload

To add page numbers to a PDF for free: drop it below, choose a format (1, 2, 3… / Page 1 of N / — 1 —), a position, and the starting number, then click Add numbers. Numbering is stamped into every page locally.

How to use this tool

  1. Drop your PDF into the tray.
  2. Pick the number format, position, and starting value.
  3. Click Add numbers and download the numbered PDF.

Required by examiners, loved by readers

Universities and courts commonly require numbered pages, and anyone assembling a merged document benefits from continuous numbering. The Page 1 of N format also guards against missing pages.

Frequently asked questions

Can numbering start from a number other than 1?

Yes — set any starting value, useful when the document continues from another file.

Which positions are available?

Bottom centre, bottom right, and top right.
